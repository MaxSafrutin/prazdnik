# Калькулятор услуг MAX SAFRUTIN

Статический проект для GitHub Pages. Работает без WordPress, PHP, сборщиков и серверной части.

## Структура

- `index.html` - разметка блоков калькулятора.
- `styles.css` - весь дизайн и адаптив, отдельно от логики.
- `app.js` - расчет, поведение полей, скидки, итоговая смета.
- `data/pricing.json` - тарифы, коэффициенты, скидки, НДС, округление.
- `data/content.json` - основные тексты и телефон.
- `assets/` - изображения и будущие медиа.

## Где что менять

- Только дизайн: `styles.css`.
- Только тексты: `data/content.json` и подписи в `index.html`.
- Только формулы и цены: `data/pricing.json` и при сложной логике `app.js`.
- Номер телефона кнопки бронирования: `data/content.json`.

## Локальный запуск

Открывать лучше через любой статический сервер, чтобы JSON-файлы загрузились корректно:

```bash
npx serve .
```

или

```bash
python -m http.server 8080
```

## GitHub Pages

1. Загрузите содержимое этой папки в корень репозитория.
2. В GitHub откройте `Settings -> Pages`.
3. Выберите `Deploy from branch`.
4. Укажите ветку `main` и папку `/root`.

## Расчет

Калькулятор считает:

- работу DJ;
- работу ведущего;
- оборудование DJ;
- реквизит ведущего;
- интернет, генератор, парковку;
- погрузку / монтаж / демонтаж;
- проезд до 300 км;
- запас 10%;
- скидки по правилам совместимости;
- юрлицо с полным НДС 22%;
- округление вниз до сотен.

Базовые ставки DJ и ведущего ежедневно увеличиваются от даты `2026-07-03`, заданной в `data/pricing.json`.
